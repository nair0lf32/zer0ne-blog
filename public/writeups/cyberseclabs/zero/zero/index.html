It’s a windows machine with RPC open on port 135. Enumeration with rpcdump.py gives no endpoint, but the nmap script says DC so we are facing a domain controller.

A simple google search for ‘domain controller vulnerabilities’ tell us about Zerologon

A critical vulnerability with a cvss of 10. It’s potentially destructive though, as it changes the DC password to blank.
